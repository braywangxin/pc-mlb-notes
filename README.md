# pc-mlb-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面以笔记、记录或工具的形式存在，不针对任何特定网站或域名。

## 项目简介

pc-mlb-notes 是一个静态页面集合仓库，存放若干独立的 HTML 文件。每个页面均为自包含的、可直接在浏览器中打开使用的文档或工具。项目不提供后端服务，不依赖外部资源，所有内容均以纯前端方式呈现。

## 目录说明

仓库根目录下直接存放各个 HTML 文件，文件名反映了页面的基本用途或主题。部分页面可能包含同名的资源文件夹（如 `assets/`、`styles/`、`scripts/`），用于存放该页面的样式或脚本文件。

- `*.html` — 独立的 HTML 页面文件
- `assets/` — 页面所需的静态资源（图片、字体等）
- `styles/` — 页面样式文件
- `scripts/` — 页面脚本文件

## 页面归档说明

每个 HTML 页面均为独立归档，内容可能随时间更新或修正。页面之间没有依赖关系，可单独访问或使用。归档内容以实用性、记录性或备忘性为主，不保证长期更新或维护。

## 维护说明

本仓库由个人维护，更新频率不固定。如发现页面内容存在错误或失效，欢迎提交 Issue 或 Pull Request。所有贡献请遵循以下原则：

- 保持页面独立性与自包含性
- 不引入外部链接或依赖
- 不添加与页面主题无关的内容

## 使用方式

可直接通过 GitHub Pages 或本地文件系统打开任意 HTML 文件。由于页面为纯前端实现，无需服务器环境。
